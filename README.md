# bd_banca_cobranzas_dsrp
Una entidad bancaria necesita gestionar y analizar la recuperación de deudas de sus clientes morosos, con el fin de: Registrar operaciones diarias de cobranza.   Controlar promesas de pago, cuotas y estados de deuda.   Evaluar la efectividad de los gestores de cobranza.   Analizar indicadores históricos para la toma de decisiones.
